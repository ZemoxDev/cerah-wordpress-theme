Cerah is a fast WordPress Theme suitable for blogs, personal portfolios and e-commerce Websites.

== Description ==
Cerah is a light colored beautiful theme with the colors orange and light grey in the foreground.
It offers an amazingly clean, modern and responsive design.
Cerah has no unnecessary parts, so it loads lightning fast which is great for the SEO optimization of your Website.

== Frequently Asked Questions ==

= How can I customize Cerah? =

    To customize Cerah you need to open the WordPress Customizer by accessing the Appearance menu -> Customizer.
